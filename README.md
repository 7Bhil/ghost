# 👻 Mirage Ghost Engine

> **Le Piège Invisible.**  
> Troisième pilier de l'écosystème **MIRAGE**, ce module est un moteur de déception capable de cloner des machines compromises pour y piéger les attaquants.

## ✨ Fonctionnalités clés (Planning)

- **Snapshot (Jour 12)** : Capture l'identité d'une machine (Bannières, Headers).
- **Clone Virtuel (Jour 13)** : Génération automatique de Honeypots Docker.
- **Redirection (Jour 14)** : Basculement transparent de l'attaquant.
- **Évolution (Jour 15)** : Faux fichiers et bases de données dynamiques.
- **Enregistrement (Jour 16)** : Logging complet des actions de l'attaquant.

## 🚀 Installation

```bash
git clone https://github.com/7Bhil/ghost.git
cd ghost
chmod +x install.sh
./install.sh
```
